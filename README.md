# Task Management REST API (Go Learning Showcase)

Go learning playground project. Minimalist task manager API built with Go & Gin.

## Stack
- Go 1.26
- Gin Web Framework
- In-memory repository (evolving)

## AI-Assisted Learning
This codebase is developed with help from AI tools:
- **Code Audits & Reviews**: AI agents perform periodic parallel reviews (security, concurrency, architecture).
- **Pair Programming**: AI assists in refactoring, error correction, and roadmap planning.
